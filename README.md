# Steps to Locally Run the Application

## Clone from  Github

### Clone the Repository:

``` bash
git clone https://github.com/sagar1952115/widgets.git
cd widgets
```
### Install Dependencies:

``` bash
npm install
``` 
### Run the Application:
``` bash 
npm run dev
```

### Open in Browser: Access the application at http://localhost:5173.
